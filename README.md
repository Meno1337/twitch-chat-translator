# 🌐 Twitch Chat Translator (Chrome Extension)

Простое и эффективное расширение для Chrome, которое автоматически переводит сообщения в чате Twitch в режиме реального времени. Идеально подходит для просмотра иностранных стримеров без языкового барьера.

## ✨ Особенности
* **Авто-перевод:** Использует Google Translate API для мгновенного перевода.
* **Умная фильтрация:** Игнорирует смайлы, цифры и слишком короткие сообщения, чтобы не засорять чат.
* **Кастомизация:** Выбор целевого языка и цвета текста перевода через всплывающее окно (Popup).
* **Native Look:** Стилизация под интерфейс Twitch для комфортного чтения.

## 🛠 Технологии
* **JavaScript (ES6+)** — логика перевода и MutationObserver для слежения за чатом.
* **Chrome Extension API (Manifest V3)** — современный стандарт расширений.
* **CSS3** — кастомная стилизация элементов перевода.

## 🚀 Установка (Developer Mode)
1. Склонируйте репозиторий: `git clone https://github.com/Meno1337/twitch-chat-translator.git`
2. Откройте Chrome и перейдите по адресу `chrome://extensions/`.
3. Включите **Режим разработчика** (Developer mode) в правом верхнем углу.
4. Нажмите **Загрузить распакованное расширение** (Load unpacked) и выберите папку с проектом.
5. Откройте любой стрим на Twitch и наслаждайтесь!

## ⚙️ Настройка
* Нажмите на иконку расширения в панели браузера.
* Выберите нужный язык (по умолчанию — Русский).
* Выберите цвет текста, который вам нравится.
* **Важно:** После изменения настроек обновите страницу стрима (F5).

## ⚠️ Дисклеймер
Проект создан в образовательных целях. Перевод осуществляется через бесплатные узлы API и может быть неточным.
