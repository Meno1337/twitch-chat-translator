# 🌐 Twitch Chat Translator (Multi-Browser Support)

Универсальное расширение для автоматического перевода сообщений в чате Twitch в режиме реального времени. Больше никакого языкового барьера при просмотре зарубежных трансляций!

---

## ✨ Особенности
* **Движок перевода:** Использует надежное Google Translate API для мгновенного результата.
* **Умная фильтрация:** Автоматически пропускает смайлы, цифры и системные сообщения, чтобы не засорять чат.
* **Кастомизация:** Выбор целевого языка и цвета текста перевода через удобное всплывающее окно (Popup).
* **Нативная интеграция:** Перевод встраивается прямо в сообщения чата, сохраняя оригинальный стиль и верстку Twitch.

---

## 📂 Структура проекта
В репозитории представлены версии, оптимизированные под разные движки:
* `📂 /chrome` — Версия для браузеров на базе Chromium (Chrome, Edge, Opera, Brave).
* `📂 /firefox` — Оптимизированная версия для Firefox (Gecko), прошедшая строгую валидацию AMO.

---

## 🛠 Технологии
* **JavaScript (ES6+)** — логика перевода и динамическое отслеживание чата через `MutationObserver`.
* **Manifest V3** — актуальный стандарт безопасности и производительности расширений.
* **CSS3** — кастомная стилизация элементов для бесшовного интерфейса.

## 🚀 Установка (Режим разработчика)

### Для Chrome / Edge / Chromium:
1. Склонируйте репозиторий: `git clone https://github.com/Meno1337/twitch-chat-translator.git`
2. Перейдите по адресу `chrome://extensions/`.
3. Включите **Режим разработчика** (Developer mode) в правом верхнем углу.
4. Нажмите **Загрузить распакованное расширение** (Load unpacked) и выберите папку `/chrome`.

### Для Firefox:
1. Перейдите по адресу `about:debugging#/runtime/this-firefox`.
2. Нажмите **Загрузить временное дополнение** (Load Temporary Add-on).
3. Выберите файл `manifest.json` из папки `/firefox`.

---

## ⚙️ Настройка
* Нажмите на иконку расширения в панели браузера.
* Выберите нужный язык (по умолчанию — **Русский**).
* Выберите цвет текста, который лучше всего читается на выбранной вами теме Twitch.
* **Важно:** После первого изменения настроек обновите страницу стрима (**F5**).
